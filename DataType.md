# Data Type in Java 
When you define any data, you have to use either variable 
or constant or literals 
to define variable 
    type identifier (name of Variable)
to define constant
    final type identifier = value (name of Variable)
examples 
```java
    int noOfStudent;
    double balanceYouOweme;
```
